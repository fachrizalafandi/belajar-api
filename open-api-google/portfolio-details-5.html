Ringkasan Produk

PLP merupakan sebuah website yang dapat memaksimalkan kinerja pengguna dalam hal pelayanan terhadap pelanggan mengenai berbagai informasi, pertanyaan, maupun keluhan yang diberikan oleh pelanggan kepada perusahaan PT. Petrokopindo Cipta Selaras. Secara garis besar, website ini dibangun menggunakan HTML, CSS Bootstrap, Javascript, dan PHP Native.
